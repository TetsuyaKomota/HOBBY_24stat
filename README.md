# 24分析ツール

## 概要
- youtube とか twitter とかいじって分析ごっこするリポジトリ

### pick_statistics.py
- 指定したチャンネルの以下の情報を取得し，tsvにまとめる
    - チャンネル名
    - 登録者数
    - 直近の投稿動画
        - 投稿日時
        - 視聴回数
        - 高評価数
        - 低評価数
        - コメント数
